# GtechFrontEnd
Projeto final com o objetivo de desenvolver uma Digital Store para conclusão do curso Geração Tech 2026.

---

## Como rodar o projeto

* **Método 1: Acessar o link do vercel https://gtech-front-8brfkdi3p-arnonluisms-projects.vercel.app**

* **Método 2: clonar o repositório, instalar as dependências (npm install) e rodar o projeto pelo terminal no localhost (npm run dev)**
